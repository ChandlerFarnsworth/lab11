# lab11 – Assignment #11: Forms, Usability & Accessibility

Accessible version of the "2020 Proposal Form" exercise from ATLS/CSCI 3150-4519.

## Files

- `forms/form-exercise1(one).html` – the form, with semantic headings, `<fieldset>`/`<legend>` grouping, `<label>`s tied to every field, and instructional text for required fields.
- `forms/css/form.css` – styling: aligned/right-justified labels, bullet removal, tighter radio-button grouping, visible keyboard/mouse focus highlighting, and color-cued required fields.
- `assignment11-report.md` – the 2-paragraph discussion write-up and live form URL for the assignment's discussion post.

## Live page

Once GitHub Pages is enabled for this repo (Settings → Pages → Deploy from branch `main`, folder `/ (root)`), the form is served at:
